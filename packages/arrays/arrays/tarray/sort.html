TArray.sort ()

macro sort (callback : variant, data : variant)

Процедура выполняет сортировку массива в соответствии с порядком, определяемым пользовательским обработчиком. В случае успешного завершения возвращает TRUE. При неудаче FASE. Причиной неудачи могут быть неверно заданные параметры.

Параметр callback принимает пользовательский обработчик сравнения двух элементов массива, имеющий прототип:

macro Compare (left: variant, right: variant, data: variant): integer

Если левый элемент больше правого необходимо вернуть значений "> 0", если меньше "< 0". Если элементы равны, необходимо вернуть "0".

Этот обработчик может быть задан строкой с именем глобальной процедуры RSL, ссылкой на процедуру, либо ссылкой на метод класса.

Параметр data передаётся из процедуры сортировки в пользовательский обработчик в качестве параметра data.
