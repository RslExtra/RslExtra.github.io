TArray

Class TArray ()

Стандартный класс TArray(). Динамический массив TArray представляет собой объектную альтернативу стандартной конструкции языка ARRAY. Имеется возможность замены конструкции ARRAY на новую конструкцию TArray:

См. пример. 

Array myArray; /* Прежняя конструкция */
myArray = TArray; /* Замена прежней конструкции на новую */
Успешно скопировано в буфер обмена

Синтаксическая форма конструктора TArray имеет следующий вид:

TArray [ ( n1 [,n2] ) ]

Память под элементы массива выделяется постранично. Параметры конструктора класса TArray задают количество элементов массива на первой странице (значение n1) и на последующих страницах (значение n2), под которые выделяется память. По умолчанию первая страница может содержать 50 элементов, все следующие страницы по 10. Эти значения можно изменить, указав параметры конструктору класса:

См. пример. 

/* Опишем массив, в котором первая страница содержит 100 элементов, а все следующие по 5:*/
MyArray = TArray ( 100, 5 );
Успешно скопировано в буфер обмена

Для удобства создания массивов передающихся в ActiveX в виде SAFEARRAY и в Java в виде Java-массивов используется конструктор TArray, который первым параметром принимает булевское значение, задающее свойство MarshalByVal для нового объекта.

В качестве параметра можно указать ссылку на уже существующий массив. В этом случае создается новый массив, содержащий копию элементов массива, обозначенного в качестве параметра.

Доступ к элементам массива осуществляется посредством указания индекса элемента в круглых скобках после переменной, ссылающейся на объект класса TArray.

См. пример. 

FirstArray = TArray;
/* Присвоение значения */
FirstArray (0) = 100;
/* Создается копия массива FirstArray */
SecondArray = TArray (FirstArray);
Успешно скопировано в буфер обмена

Объекты класса TArray имеют свойство Size, значением которого является текущий размер массива. Присваивая свойству Size новое значение, можно изменять размер массива. Увеличивается размер массива динамически при обращении к элементу, стоящему за последним элементом массива:

См. пример. 

Println ( "Текущий размер массива:", FirstArray.Size );
Успешно скопировано в буфер обмена

Объекты класса TArray имеют неименованные свойства - элементы массива, к которым можно обращаться только по индексу. Для доступа к ним предусмотрено свойство Value (N) значение N-го элемента массива.

Следует отметить, что с появлением у объектов свойств с параметрами возникла проблема, связанная с одинаковым синтаксисом указания индекса (см. стр. 6) и свойства с параметрами. Поэтому для разрешения неоднозначности предусмотрена возможность указания индекса свойства объекта в квадратных скобках.

См. примеры. 

Ob = Tarray;
Ob [5] = 100;
Пример.
/* Пример обращения к элементам массива */
class TTT /* Определяем класс ТТТ */
var prop = TArray;
prop (5) = 200;
macro Me
return prop;
end;
end;
ob = TTT; /* Создаем экземпляр класса TTT */
/* Распечатаем значение пятого элемента массива, */
/* на который ссылается свойство prop тремя способами: */
println (ob.prop() (5));
println (ob.prop .value (5));
println (ob.prop [5]);
Успешно скопировано в буфер обмена

Объекты класса TArray имеют свойство MarshalByVal типа Bool, которое определяет, каким образом объекты этого класса передаются в RSCOM. Если свойству присвоено значение TRUE, то объекты передаются по значению. По умолчанию свойство имеет значение FALSE, и, соответственно, объекты класса TArray по умолчанию передаются по ссылке.

Для удобства создания массивов передающихся в ActiveX в виде SAFEARRAY и в Java в виде Java-массивов используется конструктор TArray, который первым параметром принимает булевское значение, задающее свойство MarshalByVal для нового объекта. Вторым и третьим параметром могут передаваться количество элементов на первой странице и на последующих страницах.
