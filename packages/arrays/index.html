Определение массивов

..\mac\RslLib\StdLib\constructions\Arrays.mac

В языке RSL можно объявлять только одномерные массивы. Для этого предусмотрены следующие конструкции:

  • конструкция ARRAY;
  • стандартный класс TArray.

Примечание. При определении массивов предпочтительнее использовать класс TArray. Конструкция ARRAY в настоящее время поддерживается для совместимости с ранее разработанными программами.
