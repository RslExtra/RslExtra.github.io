StringUtilsClass.fillBetween ()

macro fillBetween (
    str1 : String,
    str2 : String,
    minLength : Integer,
    padChar : String
) : String

Возвращает строку дополненную символами padChar между строк str1 и str2 до размера minLength. Выровняет строку str1 слева, str2 справа, а промежуток заполнит символом padChar.

+

   var str1 = "left";
   var str2 = "right";
   var result = StringUtils.fillBetween(str1, str2, 12, ".");
   println(result); // left...right
