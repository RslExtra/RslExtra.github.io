StringUtilsClass.repeat ()

macro repeat (str : String, count : Integer) : String

Возвращает строку сконкатенированную count раз исходной строки str.

+

   var str1 = "word";
   var result = StringUtils.repeat(str1, 3);
   println(result); // wordwordword
