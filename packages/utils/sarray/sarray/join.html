SArray.join ()

macro join ([sep : String = ""], [pre : String = ""], [pos : String = ""]) : String

Параметры функции

sep

разделитель

pre

префикс

pos

постфикс

Объединяет элементы масcива в строку, разделяя их значением sep, значение pre - будет добавлено в начало строки, значение post в конце. Каждый из параметров по умолчанию равен пустой строке "".

Пример:

+


var arr = SArray("start", "use", "this");
println(arr.join());  // startusethis
println(arr.join(" "));  // start use this
println(arr.join(" ", "just ", "!"));  // just start use this!


import "../mac/extra/utils/SArray.mac";
