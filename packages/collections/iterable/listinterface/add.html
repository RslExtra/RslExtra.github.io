ListInterface.add ()

macro add (element : Variant, ind : Integer)

Добавляет element в список на позицию ind. Если ind не указан или равен null, добавляет элемент в конец списка.

+

list.add("second");
list.add("between", 1);

for(var l, list)
   print(l + " ");
end;
