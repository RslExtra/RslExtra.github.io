SetInterface.remove ()

macro remove (element) : Bool

Ищет и удаляет element, возвращает true, если элемент был найден и удален.

+

set.addItems(1, 2, 3);
var result = set.remove(0);
println(result);        // false
result = set.remove(1);
println(result);        // true
set.toScreen();         // [2, 3]
