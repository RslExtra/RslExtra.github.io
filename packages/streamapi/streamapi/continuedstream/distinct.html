ContinuedStream.distinct ()

macro distinct ()

Передает только уникальные элементы. Если это объектный тип, должны быть реализованы методы hash и equals.

+

import "..\\mac\\extra\\StreamApi\\Stream.mac";

var str = "This is simple text";
StreamOf(str)
   .distinct()
   .forEach(@print);
