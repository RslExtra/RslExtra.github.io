ContinuedStream.groupByInt ()

macro groupByInt (parmfunc)

Групирует элементы в структуру типа HashMap где key вычисляется функцией parmfunc. Оптимизирован для Integer ключей. Следующим элементом в цепочке будет объект типа Pair.
