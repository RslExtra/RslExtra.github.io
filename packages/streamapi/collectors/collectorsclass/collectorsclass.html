CollectorsClass

Private Class CollectorsClass ()

Предоставляет часто используемые коллекторы, представлен singleton Collectors.

Методы

counting
macro counting (keyMapper, keyType : Integer)

Возвращает коллектор в карту LinkedHashMap, которая групирует по ключу возвращаемому keyMapper, типа keyType, и значением как количество элементов соответсвующих ключу

toArray
macro toArray ()

Возвращает коллектор в массив SArray

toLinkedList
macro toLinkedList ()

Возвращает коллектор в список LinkedList

toList
macro toList ()

Возвращает коллектор в список ArrayList

toMap
macro toMap (keyMapper, valueMapper)

Возвращает коллектор в карту HashMap согласно преобразованию ключей через функцию keyMapper и значений valueMapper

toTArray
macro toTArray ()

Возвращает коллектор в массив TArray
