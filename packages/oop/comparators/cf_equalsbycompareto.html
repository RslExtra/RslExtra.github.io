CF_EqualsByCompareTo ()

macro CF_EqualsByCompareTo (o1 : Object, o2 : Object) : Bool

Проверка равенства двух экземпляров пользовательского класса, реализующих метод compareTo(otherObj) : Integer. Возвращает true если compareTo вернет 0. Способен принимать null в качестве аргументов, однако результат сравнения с null должен быть определен в Equals
