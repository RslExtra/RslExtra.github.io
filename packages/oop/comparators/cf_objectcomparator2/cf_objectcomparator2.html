CF_ObjectComparator2

Class CF_ObjectComparator2 (obj)

Объект-компаратор генерирует метод-компаратор и предоставляет ссылку на него методом getFunc. Схож с CF_ObjectComparator, однако сравнивает объекты одного класса, в конструктор он передается первым параметром obj. Далее следуют имена полей, в т.ч и приватных. Если поле с таким именем не найдено у любого из объектов, будет брошена ошибка.

Методы

compare
macro compare (o1, o2) : Integer

Возвращает результат сравнения двух объектов

getFunc
macro getFunc ()

Возвращает ссылку на метод compare
