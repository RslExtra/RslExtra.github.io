Функции

DebugBreak
macro DebugBreak ()

Если в системе установлен отладчик RSL, то эта процедура вызывает прерывание исполнения RSL программы и активизацию отладчика на следующей после DebugBreak инструкции

ExecExp
macro ExecExp (string) : Variant

Процедура возвращает результат вычисления выражения string

ExecMacro
macro ExecMacro (string) : Bool

Данная процедура вызывает для выполнения процедуру, заданную в качестве первого параметра string

ExecMacro2
macro ExecMacro2 (string) : Variant

Процедура вызывает процедуру string и возвращает значение, которое возвращает запускаемая процедура

RunError
macro RunError (mes : String, userObj : Variant)

Вызывает аварийной завершение RSL программы
